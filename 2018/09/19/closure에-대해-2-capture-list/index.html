2018-09-19
#swift

capture list에 대해 araboza

capture란?

클로저의 본문에서 인스턴스의 프로퍼티에 접근하거나 인스턴스의 메소드를 호출하는 것을 캡쳐(capture)라고 한다.

Value Type Capture

위의 코드를 실행하면

초기화 값 : 1 2 10 0

a, b 만 캡쳐

변경 값 : 10 10 5 25

클로저 내부 값 : 1 2 5 8

실행 이후 값 : 10 10 5 8 이 된다.

차례대로 보면 valueCapture 에서 a,b를 캡쳐할때 위의 1,2가 캡쳐되어 저장된 상태가 된다. 그래서 밑에서 valueCapture()를 실행하고 그 안에서 print를 찍으면 먼저 캡쳐된 1,2 값이 저장되어 나온다. 값을 변경하고 valueCapture를 실행했기 때문에 내부값은 캡쳐된 1,2와 변경된 c = 5, result = a + b + c = 8 이 되었고, 실행 이후값은 a,b,c는 10,10,5가 되고 result는 valueCapture()안에서 변경되었기 때문에 8이 된다!

Reference Type Capture

위의 코드 실행 결과는

초기화 값 : 0 0

변경 값 : 5 7

클로저 내부 값 : 5 7

실행 이후 값 : 5 7

변경 값 : 5 7 이 된다.

위의 valueCapture와는 다르게 값이 아닌 주소값을 캡쳐한다.
그래서 값을 변경해도 변경한 값이 캡쳐한 x에 들어가기 때문에
클로저 내부값과 실행 이후 값이 변경 값인 5,7이 되는것이다.
